# README.md


Система отслеживания задач "Stacklog"

**Назначение:** Автоматизация управления циклом разработки и модификации систем.

**Стек технологий:**
- **Backend:** .NET 10 (C#)
- **Database:** PostgreSQL (основная), Redis (кеш)
- **Frontend:** React JS + TypeScript
- **Архитектура:** Clean Architecture / Monolith (с заделом под микросервисы)
